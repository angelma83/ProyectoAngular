En este punto se explicará la creación de patrones o pipes que vimos durante el curso, recordad que algunos nos los proporciona Angular.

Vimos varias opciones durante el curso, como por ejemplo las fechas, los números, monedas

Para obtener información, https://angular.io/guide/pipes.

Patrones en Fechas, decimales, mayúsculas

Se muestra el valor de fecha sin patrón o pipe {{fecha}}

Valor con patrón o pipe, se recuerda debe incluir | {{fecha | date:'d/M/y H:m'}}

Patron mayúscula y minúscula, uppercase y lowercase para la variable ciudad, valor de variable, {{ciudad}}, con las pipes {{ciudad | uppercase}}, {{ciudad | lowercase}}

Se incluye patrón o pipes de números, variable resultado tipe number, valor de variable {{numero}}, con pipe {{numero | number:'3.2-3'}}
